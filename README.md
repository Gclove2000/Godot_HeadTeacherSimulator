# Godot_HeadTeacherSimulator
Godot + C# 班主任模拟器
# 游戏玩法

你作为一个年级班主任，你将管理一群学生（暂定24人），带领他们准备一年的高中学习。

# 游戏基础

- 一年两个学期，一个学期4个半月，18个星期
- 考试分为月考，开学考，期中考，期末考。
  - 月考：4，8，12，16
  - 开学考：1
  - 期中考：10
  - 期末考：18
- 考试分为语文，数学，英语
- 一周精简为一天，分为上午四节，下午四节，
  - 英语：2
  - 语文：2
  - 数学：2
  - 体育：1，必定最后第二节
  - 班会：1，必定最后一节
